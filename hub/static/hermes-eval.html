Verdict

Use Hermes to burn idle free capacity on bounded, reversible work: summaries, first-pass code review, triage, test-case ideas, small policy drafts, and background research briefs.

Do not hand it production edits, secrets work, external actions, or ambiguous repo automation without a stronger agent reviewing the output.

Best Uses

Where Hermes should spend idle free quota
Good fits
  • First-pass issue triage, duplicate detection, and acceptance-criteria cleanup.
  • Read-only repo reconnaissance with explicit snippets or file lists.
  • Drafting test cases, review checklists, runbooks, and changelog summaries.
  • Nightly or heartbeat background briefs that are cheap to discard.
  • Second opinions on simple bugs where the answer can be verified locally.
Avoid
  • Autonomous code edits or deployments.
  • Tasks involving secrets, OAuth, payments, or external messages.
  • Open-ended architecture decisions without source context.
  • Pinned free-model dependence unless the model is rechecked that day.
  • Anything where hallucinated local process names would cause damage.

Operating Rules

Conservative defaults for Nest
Default route

Prefer openrouter/free for generic background tasks. It was more reliable than pinned free models in this run.

Pinned route

Use pinned models only for isolated experiments or when today's provider availability has been checked.

Budget

Keep bursts under 20 RPM. The daily benchmark may use up to an estimated 66% of currently available free requests, leaving roughly a third untouched.

Review

Treat Hermes output as a draft. OpenClaw/Codex or the human validates anything that changes state.

Research Notes

Checked 2026-06-10
  • OpenRouter documents free :free models at 20 requests/minute, with 50/day before $10 purchased credits and 1000/day after that: OpenRouter limits.
  • Hermes' OpenRouter integration supports openrouter/free, pinned model IDs, provider routing, fallback providers, and auxiliary model routing: Hermes Agent integration.
  • OpenRouter's model API can list current free model IDs, context windows, pricing, and supported parameters: Models API.
  • OpenRouter usage accounting is returned in API responses, but Nest's current Hermes ledger source only counts gateway log responses, not CLI one-shots: Usage accounting.
